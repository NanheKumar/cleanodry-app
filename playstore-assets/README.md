# Cleanodry Play Store Assets

This folder is for Google Play listing assets and release text.

Image files are intentionally not included yet. Do not add empty, fake, AI-generated, or placeholder PNG files. Add only final assets captured from the real Android app or approved Cleanodry design source files.

## Required Image Checklist

- [ ] `app-icon-512.png` - 512x512 PNG app icon
- [ ] `feature-graphic-1024x500.png` - 1024x500 PNG feature graphic

## Screenshot Requirements

- Use portrait Android screenshots from the production-ready app.
- Do not show test credentials, staging URLs, debug banners, private data, or broken UI.
- Use consistent device framing, theme, language, and account state across screenshots.
- Use a safe demo/customer account with non-real personal data.
